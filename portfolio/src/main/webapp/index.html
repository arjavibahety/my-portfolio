Projects

YouTube Cat

Featured at Google Developer Space January 2020 La Kopi

People usually have a love-hate relationship with YouTube. While it provides a plethora of educational content, it can be a very influencial source of distraction which results in a huge pile of homework :( YouTube Cat is a Chrome browser productivity extension which works on YouTube to block videos of selected categories. I developed it with a schoolmate over the summer break as part of NUS's independent software engineering module called "Orbital".
The full code can be found in the Github repository YouTube-cat
More information about our talk at La Kopi can be found here.

Hackomania 2019

Best Rakuten API ‘API Hack’ Award

I took part in HackOmania which is an annual 24-hour hackathon organized by the GeeksHacking Community in Singapore. Rakuten API challenged hackathon participants to use their APIs to address the problem statements. My team decided to work on a mobile application which gamifies the way diabetic patients keep track of their food intake. As the backend engineer, I connected the frontend to the various Rakuten APIs and used Firebase to store the user data. We managed to win the First Prize in the RakutenAPI “API Hack” category. It was my first-time using C# and Unity to make API calls and it was a truly enriching experience.
The full code can be found on our Github repository Pokkamon
You can read more about the app here.

MRT Game

First hackathon!

I took part in "HacknRoll" - an annual 24 hour hackathon organised by NUS Hackers. We all take the MRT. It's a ubiquitous part of Singaporean culture. The MRT is a hotchpotch of Singaporean diversity. The story of the MRT is one of culture and public etiquette. The objective of this game is to get a seat on the MRT before everyone else does! It portrays Singapore's hectic lifestyle too. I worked on the movement of the player using python and object-oriented programming. I also helped in coding the movement of the NPCs as well as combining the code contributed by my team members.
The full code can be found in the repository MRTgame
